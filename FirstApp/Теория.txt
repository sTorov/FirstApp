В качестве ограничений для параметра могут выступать:

- Определённый класс, используя его наименование.
- Определённый интерфейс, используя его наименование (об интерфейсах мы поговорим в одном из следующих модулей).
- Любой класс, используя ключевое слово class.
- Любая структура, используя ключевое слово struct.
- Любой тип с конструктором без параметров, используя new().

Определённый класс	
class Order<T> where T : Delivery

Любой класс	
class Order<T> where T : class

Любая структура	
class Order<T> where T : struct

Любой тип с конструктором без параметров	
class Order<T> where T : new()
